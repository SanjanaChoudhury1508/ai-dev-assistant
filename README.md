# AI Developer Assistant (Terminal UI)

A terminal-style multi-session developer assistant built with Next.js and Tailwind CSS.

## Features

* Multi-session chat system
* Command support (/clear, /help)
* LocalStorage persistence
* Typewriter effect responses
* Keyboard shortcuts
* Clean terminal UI

## Tech Stack

* Next.js
* React
* Tailwind CSS
